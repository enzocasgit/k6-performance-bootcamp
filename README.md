# k6-performance-bootcamp
| Description |
Repo for storing k6 performance testing scripts that will be used on Atlantico's Avanti Bootcamp.

| How to use |
Firstly, install K6 Framework on Grafana's Website (https://grafana.com/docs/k6/latest/set-up/install-k6/)
Then, run the script on your local terminal.
-> If on Windows:
- Open CMD
- Open the folder where the script is installed
- Type "k6 run load.js"

-> Linux:
- Open Terminal
- Open the folder where the script is installed
- Type "k6 run load.js"
